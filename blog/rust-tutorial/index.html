Free Standing kernel

First we disabled standard library (we cannot use any os dependent libraries ).Also we will need panic implementation

Panic implementation it called when ever something fucks up so we save memory and time by aborting the compile and allocation of the memory for the failed or buggy function

Speaking of compiler and runtime we also need to be careful with compilation errors There are 3 types of errors:

  • compiler warnings;(Don't ignore but tbh it isn't something severe enough to keep your code from compiling, unless you instruct your compiler to treat them as errors )

  • compiler errors;(on the other hand we have errors they are conditions that prevent the compiler from compilation of the file or code )

  • linker errors;( Don;t care for syntax , care for finding definition for functions ,classes etc e.g could not find definition x)

Why we need to think of linker errors , its because of different os like Linux , win etc... we could use rust bare metal target it automatically tries to detect what type of executable is it ".exe" or something else
